# STYLE 08 — 럭셔리 다크 (매력 축: 품격)

매력 가설: 프리미엄의 언어는 "느림"이다. 빠르고 튀는 것은 싸 보인다.
고도화 = 호흡을 늦추고 소재를 올리는 것. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 럭셔리 다크 v2]

■ 무드
프라이빗 뱅킹 라운지. 어두운 월넛과 골드 헤어라인, 낮은 조도.
대시보드가 아니라 "오늘의 자산 리포트"를 받아드는 감각.

■ 팔레트
배경 #0E0D0B / 패널 #171512 / 골드 #C9A96A / 밝은 골드 #E5CFA3(극소량) /
아이보리 #EDE8DF / 보조 #8A8477 / 하락·문제 #B0665C(채도 낮은 적갈 — 원색 빨강 금지)
골드는 보더·괘선·수치 강조에만. 골드 면적이 화면의 5%를 넘으면 실패다.

■ 타이포 — 이 시안의 주인공
수치·제목: Cormorant Garamond, KPI 수치 48px 이상, 굵기 500 — 가늘고 큰 것이 품격
레이블: Pretendard 11px, 자간 .12em, 대문자/소형 대문자 느낌
세리프·산세리프 대비 자체가 장식이므로 다른 장식 불필요.

■ 깊이 — 그림자가 아니라 "조명"
그림자 금지. 패널 상단에 rgba(201,169,106,.06) 그라디언트를 얹어
위에서 조명이 떨어지는 소재감을 만든다. 구획은 1px 골드 헤어라인(opacity .25)이 전담.

■ 히어로 연출 — "장이 열린다" (총 2.0초, 느리게)
1) 0.0s: 어두운 배경 위에 골드 헤어라인들이 좌→우로 그려진다 (선이 먼저, 내용은 나중)
2) 0.6s~: KPI 수치가 순차 페이드인 — 카운트업 금지 (숫자가 뛰는 것은 품격에 어긋난다)
3) 1.2s~: 차트 막대가 700ms ease-out으로 차오른다. 바운스·오버슈트 절대 금지
4) 2.0s: 피크 막대 상단에만 밝은 골드 캡 점등
모든 이징은 ease-out 단일. 모든 개별 모션 500ms 이상 — 느림이 곧 연출이다.

■ 마이크로 인터랙션 (300~400ms의 느린 반응)
1) KPI 카드 hover: 배경 반 톤 상승(#1C1915) + 헤어라인 골드 선명. 이동·부유 금지
2) 차트 막대 hover: 막대 위에 세리프 이탤릭 수치가 조용히 페이드인
3) 테이블 행 hover: 행 하단 골드 헤어라인이 좌→우로 그어진다

■ 시그니처
그려지는 골드 선 + 세리프 대형 수치 — 리포트에 만년필로 밑줄 긋는 감각.

■ 접근성·안전장치
- prefers-reduced-motion: 전 연출 제거, 완성 상태 렌더
- 골드-온-다크 대비 부족 지점은 아이보리 텍스트가 본체 담당 (골드는 장식)

■ 금지
바운스, 카운트업, 글로우/네온, 이모지, 알약 태그(상태는 소형 대문자 텍스트+색점),
그림자, 원색, 300ms 미만 모션, 굵기 700 이상 세리프

■ 자체 크리틱 루프
1) 화면에서 가장 "싸 보이는" 요소 1개는? → 제거하거나 골드 헤어라인 문법으로 교체
2) 골드 면적이 5%를 넘는가? → 줄여라. 골드는 희소해야 금이다
3) 500ms 미만의 모션이 하나라도 있는가?
4) 세리프 수치가 화면의 주인공인가? 아니라면 다른 요소를 낮춰라
```
