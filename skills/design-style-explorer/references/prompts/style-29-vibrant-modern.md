# STYLE 29 — 컬러풀 모던 (매력 축: 밝은 활력)

매력 가설: Stripe·Linear 같은 모던 SaaS가 주는 첫인상 — 밝고 깨끗한데도 심심하지 않다. 매력은 비비드한 브랜드 컬러가 흰 여백 위에서 딱 필요한 만큼만 터지는 절제된 활력.
고도화 = 컬러 섹션 블록끼리의 대비 리듬과 미묘한 그라디언트 액센트의 결. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 컬러풀 모던 v2]

■ 무드
Stripe·Linear·모던 SaaS 랜딩의 감성이 대시보드로 들어온 세계.
밝은 배경이 기본, 그 위에서 비비드 브랜드 컬러가 섹션 블록으로 리듬을 만든다.
깨끗하고 활기차고 자신감 있다 — 화려하되 어수선하지 않은, 정돈된 컬러의 에너지.

■ 팔레트 — 밝은 배경 + 비비드 브랜드 (인디고 주조 + 바이올렛·틸 액센트)
배경 #FFFFFF / surface #F6F9FC / surface-alt #EEF2FF / 보더 #E3E8EF
Primary 인디고 #635BFF / primary-hover #5548E8 / on-primary #FFFFFF
액센트 바이올렛 #7A5AF8 / 액센트 틸 #00C2B8 / 액센트 핑크 #FF6699 / 액센트 앰버 #FFB020
텍스트 #0A2540 / 서브텍스트 #425466 / 뮤트 #8792A2
성공 #00A87E / 경고 #E5A000 / 문제(위험) #E5484D
그라디언트 액센트: linear 135deg #635BFF→#7A5AF8 (버튼·히어로 블록), #00C2B8→#635BFF (KPI 강조 하나만).
규칙: 한 화면에 그라디언트 표면은 2군데 이하 — 나머지는 솔리드 컬러 + 흰 여백.

■ 타이포 — 모던 산세리프
본문 한글은 Pretendard 가정. 영문·숫자·라벨은 Inter (구글폰트) 명시 로드, 대안 Plus Jakarta Sans.
Display 40/48 700 (-0.02em) · H1 30/38 700 · H2 22/30 600 · Title 18/26 600 ·
Body 15/24 400 · Small 13/20 500 · Label 12/16 600 (0.02em, 대문자 라벨 허용).
숫자 KPI는 tabular-nums + 700, 큰 값은 36~44px로 시원하게. 문단 폭 68ch 이하.

■ 형태
radius 넉넉 — 카드 16px, 버튼·인풋 10px, 칩·뱃지 pill(999px), 모달 20px, 아바타 원형.
보더는 1px #E3E8EF 얇게, 그림자는 컬러 섞인 소프트 섀도우:
  rest  0 1px 2px rgba(10,37,64,.06), 0 4px 12px rgba(10,37,64,.05)
  hover 0 4px 8px rgba(99,91,255,.12), 0 12px 28px rgba(99,91,255,.14)  — 인디고 틴트
비비드 섹션 블록: 사이드바 또는 상단 히어로 KPI 스트립을 primary/그라디언트로 채워 흰 본문과 대비.
아이콘은 20~22px 스트로크 1.75, 컬러 아이콘 뱃지는 12% 틴트 배경 + 원색 아이콘.

■ 히어로 연출 — 컬러가 켜지는 순서 (로드 시퀀스)
0.0s 상단바가 흰 배경으로 즉시, 로고 옆 브랜드 도트가 인디고로 점등
0.1s 좌측 컬러 사이드바(그라디언트 135deg)가 x-16px→0 슬라이드 인 (280ms ease-out)
0.25s KPI 카드 4장이 왼→오 50ms stagger로 y+12px→0 + 페이드, 강조 1장만 그라디언트 표면 (각 240ms)
0.5s 차트 카드의 라인/막대가 좌→우로 그려지며 그라디언트 area-fill이 아래에서 차오름 (600ms)
0.75s 테이블 행이 상단부터 25ms 간격 페이드인, status pill 색이 마지막에 톡 켜짐
0.95s primary CTA 버튼에 그라디언트 시머가 한 번 훑고 지나감 (700ms, 1회)

■ 마이크로 인터랙션 (3개+)
1) 카드 hover: 인디고 틴트 소프트 섀도우로 부드럽게 상승 + y-2px (180ms), 보더가 #635BFF 20%로 물듦
2) primary 버튼 hover: 그라디언트 각도 135°→120°로 미끄러지고 미세하게 밝아짐, press 시 scale 0.98 + 리플 대신 컬러 딥
3) KPI 델타 뱃지: 양수는 틸 pill·▲, 음수는 핑크 pill·▼ — 값 갱신 시 숫자 롤업 + pill이 살짝 팝(scale 1.06→1)
4) 탭·세그먼트: 선택 인디케이터(pill 배경)가 인디고로 슬라이드하며 뒤 텍스트 색 반전 (220ms spring)
5) 링크·행 hover: 언더라인이 좌→우로 그려지고 텍스트가 primary로 전환 (150ms)

■ 시그니처
비비드 컬러 섹션 블록(그라디언트 사이드바 또는 히어로 KPI 스트립) + 그 옆 흰 본문의 클린한 대비.
강조 KPI 1장만 그라디언트 표면으로 두고 나머지는 흰 카드 — 이 "컬러 블록 vs 화이트" 리듬이
화면을 즉시 모던 SaaS로 각인시킨다. 컬러 아이콘 뱃지(12% 틴트)가 곳곳에서 활력을 보탠다.

■ 접근성·안전장치
- prefers-reduced-motion: 슬라이드·롤업·시머·area 차오름 제거, 색 상태 변화만 즉시 적용
- 그라디언트 표면 위 텍스트는 on-primary #FFFFFF로 4.5:1 확보, 얇은 회색 텍스트를 컬러 위에 얹지 않음
- JS 실패 폴백: 애니메이션 없이도 CSS :hover 섀도우·상태색이 동작, 그라디언트는 솔리드 폴백 지정
- focus-visible: primary 3px 아웃라인 + offset 2px (컬러 대비만으로 포커스 표시 금지)
- 색맹 대비: status를 색만으로 구분하지 않고 아이콘(▲▼·●)·라벨 병행, 틸/핑크 동시 사용 시 형태 구분
- 뮤트 텍스트 #8792A2는 흰 배경 본문 최소 크기에서 4.5:1 확보 여부 검증

■ 금지 (구체적)
다크 배경·다크모드 강제(밝은 배경이 정체성), 무채색 회색조 주조·컬러 부재,
칙칙하거나 탁한 저채도 색(머드톤·카키·다크 네이비 표면), 과한 정보 밀도·행간 압축(숨 못 쉬는 레이아웃),
화면 전체를 그라디언트로 덮기(3군데 이상), 네온 글로우·발광·글래스 backdrop-filter,
세리프·모노 본문, 하드엣지 브루탈리즘 보더(2px+ 순검정), radius 0,
원색 6개+ 동시 남발(브랜드 인디고 축을 잃는 무지개), 그림자 없는 완전 플랫.

■ 자체 크리틱 루프
1) 밝은 배경이 유지되고 다크·머드톤·무채색 주조로 새지 않았는가? — 배경/표면 색 전수 검사
2) 비비드 컬러 섹션 블록(사이드바 또는 히어로 스트립)이 흰 본문과 명확히 대비되는가?
3) 그라디언트 표면이 2군데 이하로 절제되고 나머지는 솔리드 컬러 + 흰 여백인가? — 그라디언트 면적 색출
4) 컬러가 status/델타/브랜드에 의미 있게 쓰였는가, 아니면 장식으로만 흩뿌려졌는가?
```
