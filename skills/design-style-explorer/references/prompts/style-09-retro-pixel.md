# STYLE 09 — 레트로 픽셀 (매력 축: 향수)

매력 가설: 게임을 켜던 순간의 설렘. 고도화 = 게임 문법의 정확한 인용.
마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 레트로 픽셀 v2]

■ 무드
8비트 게임 콘솔. 이 대시보드가 게임 UI라면 — 데이터는 스코어, 화면은 스테이지.

■ 팔레트
배경 #1A1C2C / 패널 #29366F / 그린 #38B764 / 옐로 #FFCD75 / 레드 #B13E53 /
텍스트 #F4F4F4 (저채도 레트로 톤 — 형광 네온 금지, 유채색 6색 이하)

■ 타이포
제목·수치: 'Press Start 2P' — 작게 절제 (제목 14px 이하, 수치 20px 내외)
한글: 'DungGeunMo'(CDN 로드 가능 시) 또는 Pretendard — Press Start 2P로 한글 금지

■ 형태·질감
픽셀 계단식 보더(다중 box-shadow로 구현), radius 0, 4px 단위 그리드,
디더링 패턴 배경(반복 그라디언트), 게임 아이템 창 스타일의 상태 뱃지.

■ 히어로 연출 — "스테이지 시작" (총 1.5초)
1) 0.0s: 어두운 화면 중앙에 "READY?" 텍스트가 2회 점멸
2) 0.6s: 패널들이 즉시 스냅 등장 — 페이드 아님. 픽셀 세계에 안티앨리어싱은 없다
3) 0.8s~: 차트가 픽셀 블록 단위로 계단식으로 쌓인다 (부드러운 성장 금지, 블록 단위 스텝)
4) 1.4s: 피크 막대 위에 ▶ 커서 점멸 시작

■ 마이크로 인터랙션
1) 테이블 행 hover: 행 좌측에 ▶ 선택 커서 등장 (게임 메뉴 문법)
2) 카드 hover: 보더 색 반전 (안쪽↔바깥쪽)
3) 상태 뱃지: HP바 느낌의 블록 게이지로 (문제 상태는 빨간 블록 점멸)

■ 시그니처
READY? 오프닝 + 블록 단위로 쌓이는 차트.

■ 접근성·안전장치
- prefers-reduced-motion: READY? 생략, 점멸 정지, 완성 상태 렌더
- Press Start 2P의 가독 한계 — 문장은 반드시 한글 폰트로

■ 금지
그라디언트(디더링 패턴 제외), 블러, 부드러운 곡선, 페이드 전환(모든 전환은 스냅),
세리프, 실사풍 그림자, 16px 초과 Press Start 2P

■ 자체 크리틱 루프
1) 곡선·안티앨리어싱 느낌이 남은 곳이 있는가? → 제거
2) Press Start 2P가 가독성을 해치는 곳은? → 한글 폰트로 교체
3) 게임 문법이 아닌 장식이 있는가? → 게임에 있는 것만 남겨라
4) 유채색이 6색을 넘는가?
```
