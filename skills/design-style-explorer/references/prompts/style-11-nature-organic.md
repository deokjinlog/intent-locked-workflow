# STYLE 11 — 네이처 오가닉 (매력 축: 온기)

매력 가설: 편안함. 화면에서 흙과 종이 냄새가 나면 마음이 풀린다.
고도화 = 질감과 "성장"의 은유. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 네이처 오가닉 v2]

■ 무드
지속가능성 리포트. 흙·잎·종이. 차가운 데이터를 따뜻하게 감싼다.

■ 팔레트
배경 #F4F1E8 / 딥그린 #2D4A3E(주) / 세이지 #8BA888 / 테라코타 #C4704B(경고·문제) /
잉크 #2B2A26 — 순수 흰색·검정 금지, 모든 색은 따뜻한 톤으로

■ 타이포
제목: Gowun Batang(또는 부드러운 세리프) / 본문: Pretendard 400~600 (따뜻한 굵기)

■ 형태·질감
비대칭 유기 radius — 단 규칙을 가질 것 (예: 24px 40px 24px 40px 일관 적용),
미세한 종이 질감(SVG 노이즈 배경, opacity .03), 1.5px 손그림 느낌 보더.

■ 히어로 연출 — "자라난다" (총 1.5초)
1) 0.0s: 배경·헤더가 먼저 자리잡는다
2) 0.3s~: 차트 막대가 식물 줄기처럼 700ms ease-out으로 자란다 (끝은 둥글게),
   피크만 세이지→딥그린으로 짙어진다
3) KPI는 부드럽게 페이드인
바운스 금지 — 식물은 튀지 않는다.

■ 마이크로 인터랙션
1) 카드 hover: 그림자 대신 배경이 한 톤 따뜻해진다
2) 차트 막대 hover: 끝에 작은 잎(원형 캡)이 점등
3) 상태 표현: 씨앗(대기)→새싹(진행)→열매(완료)→시듦(문제)의 색 은유
   — 아이콘이 아니라 색 단계로만. 은유가 과하면 유치해진다

■ 시그니처
줄기처럼 자라는 차트 + 종이 질감.

■ 접근성·안전장치
- prefers-reduced-motion: 성장 연출 제거, 완성 상태 렌더
- 질감 노이즈가 텍스트 가독을 해치지 않게 opacity .03 이하 유지

■ 금지
네온, 순백/순흑, 직각, 기계적 모노스페이스, 빠른 모션(500ms 미만),
강한 그림자, 차가운 파랑, 과도한 자연 아이콘(잎사귀 일러스트 남발)

■ 자체 크리틱 루프
1) 질감이 "노이즈"로 보이는가 "종이"로 보이는가? → opacity 조절
2) 화면의 모든 색 온도가 따뜻한가? 차가운 색이 섞였는가?
3) 유기 radius가 일관된 규칙을 갖는가? (무작위≠유기적)
4) 은유가 과해서 유치해진 곳은? → 덜어내라
```
