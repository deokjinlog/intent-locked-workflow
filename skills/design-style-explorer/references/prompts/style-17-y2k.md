# STYLE 17 — Y2K 베이퍼웨이브 (매력 축: 복고미래 크롬/과잉)

매력 가설: 2000년대 웹이 상상한 "미래"는 전부 크롬으로 도금돼 있었다 — 액체 금속 로고, 홀로그램 스티커, 무한히 뻗는 네온 격자. 지금 보면 촌스러운 그 미래상이 오히려 짜릿하다.
매력은 절제가 아니라 과잉이다. 크롬 텍스트·홀로그램 그라디언트·신스웨이브 원근 격자·스캔라인을 한 화면에 겹쳐 쌓는 뻔뻔함이 정체성이다. 이 스타일은 "금속이 빛에 미끄러지는 순간"과 "지평선으로 소실되는 격자"를 동시에 보여줄 때만 완성된다. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — Y2K 베이퍼웨이브 · 복고미래 크롬 v2]

■ 무드
밀레니엄 전환기 웹 + 신스웨이브 나이트드라이브. 크롬 메탈로 각인된 미래,
자정의 퍼플 하늘 아래 지평선에서 무한히 뻗어 내려오는 네온 격자. 세련보다 강렬.
표면은 반드시 젖은 금속처럼 반사한다 — 무광(matte)은 이 스타일의 죽음이다.
얌전한 순간 실패. 모든 텍스트·보더·배경은 "빛을 받는 재질"이어야 한다.

■ 팔레트
배경 그라디언트: 상단 #1A0B2E → 중단 #2D0B45 → 하단 #0B0221 (다크 퍼플 심야, radial+linear 겹침) /
홀로그램 축(iridescence): 마젠타 #FF2FB9 ↔ 시안 #22E1FF ↔ 퍼플 #A24BFF ↔ 라임빛 시안 #66FFE3 —
  네 색을 conic 또는 linear 그라디언트로 순환시켜 보는 각도마다 색이 바뀌는 홀로그램 필름 감각 /
네온 하이라이트 핑크 #FF6EC7 / 그리드 라인 #FF2FB9~#22E1FF 그라디언트(지평선에서 시안, 근경에서 마젠타) /
크롬 텍스트 그라디언트(금속 반사 6스톱, 반드시 하이라이트-미드-하이라이트가 두 번 이상 교차):
  #FFFFFF 0% → #C7D6EA 18% → #6B7A99 40% → #EFF6FF 58%(2차 하이라이트) → #9FB4CC 78% → #E7EEF7 100% /
크롬 배경 판(chrome bar/버튼): #B8C6DA → #5C6B87 → #E7EEF7 (수평 금속 밴드) /
문제/경고 #FF3B6B / 성공 #3BFFB0 (성공도 네온 발광 — 무광 초록 금지)
색은 겹칠수록·미끄러질수록 좋다. 무채색 절제·단일 불투명 원색은 이 스타일의 죽음.

■ 타이포
디스플레이(헤더·KPI 숫자·차트 라벨): Orbitron 700~900 또는 Chakra Petch 600,
  italic 살짝 기울임(-6deg skew 또는 font-style italic) + 크롬 그라디언트 클립 + 얇은 어두운 드롭섀도(text-shadow 0 1px 0 #2A3450)로 금속 각인(engraved) 느낌.
한글 본문·표: Pretendard 400~600 (미래틱 폰트는 라틴/숫자 전용, 한글은 가독성 우선).
헤더 텍스트는 letter-spacing 0.08em + uppercase(라틴) — 디지털 전광판/LCD 감각.
숫자는 tabular-nums 고정폭 — 계기판처럼 자릿수가 흔들리지 않게.

■ 형태
카드: 반투명 다크 퍼플 #2D0B45 at 60% + 1px 홀로그램 그라디언트 보더(마젠타→시안→퍼플 순환).
  radius 14px, 외곽 네온 글로우 box-shadow: 0 0 24px rgba(255,47,185,.35), inset 0 1px 0 rgba(255,255,255,.12)(상단 금속 하이라이트 라인).
  모서리에 작은 크롬 브래킷(⌐ ⌐ 꺾쇠) 또는 홀로그램 스티커 원형 배지 장식 허용(과하게).
  배경엔 항상 스캔라인 오버레이(반복 2px 가로줄, opacity .05~.08) — CRT/전광판 질감을 상시 유지.
버튼: 크롬 배경 판 + 1px 라이트 하이라이트 보더 + 눌리면 금속 반전. 무광 단색 버튼 금지.
구획선/디바이더: 홀로그램 그라디언트 헤어라인(1px). 밋밋한 회색 실선 금지.

■ 히어로 연출 — 로드 시퀀스
0.0s: 화면 암전(#0B0221)에서 신스웨이브 그리드가 지평선(수직 60% 지점)에서 아래로 원근 전개 —
  perspective 격자가 800ms에 걸쳐 화면 하단까지 깔린다(지평선 쪽은 시안, 근경은 마젠타로 라인 색이 변함).
0.3s: 그리드 위로 크롬 타이틀이 스캔되듯 위→아래 리빌(clip-path 또는 mask 스윕, 600ms)
  + 금속 그라디언트 하이라이트(specular highlight)가 좌→우로 1회 훑고 지나감 — 빛이 금속면을 미끄러지는 연출.
0.9s: KPI 4장이 시차 120ms 간격으로 아래에서 튀어오르며 네온 글로우 점등(각 400ms), 상단 하이라이트 라인이 먼저 켜짐.
1.5s: 우상단·좌하단에 별/반짝임 파티클(✦ ✧)이 랜덤 위치에서 트윙클 시작(무한 루프) + 지평선에서 은은한 sun-glow(수평 발광 띠).

■ 마이크로 인터랙션
1) 카드 hover: 홀로그램 보더 그라디언트가 각도 회전(--angle 0→360deg, 1.2s), 글로우 반경 +8px 확대.
2) KPI 숫자 hover: 크롬 그라디언트 하이라이트 스윕이 좌→우로 재생(500ms) — 금속에 빛이 미끄러지는 감각.
3) 차트 막대/라인 hover: 마젠타↔시안 글로우 펄스(700ms) + 값 툴팁이 네온 보더로 팝.
4) 테이블 행 hover: 좌측 2px 홀로그램 바 점등 + 행 배경 rgba(162,75,255,.12).
5) 버튼 active: 크롬 표면이 반전(그라디언트 방향 뒤집힘) + 짧은 글리치 시프트(2px, 80ms) + 스캔라인 순간 밀림.

■ 시그니처
크롬 그라디언트 텍스트(금속 반사 6스톱, 하이라이트 2회 교차, background-clip:text)로 찍힌 타이틀 +
화면 하단을 가로지르는 신스웨이브 원근 그리드 지평선(마젠타→시안 라인, 지평선으로 소실) +
전면에 항상 깔린 미세 스캔라인 오버레이 + 각도에 따라 색이 변하는 홀로그램 보더/배지.
이 넷(크롬 텍스트 · 원근 격자 지평선 · 스캔라인 · 홀로그램)이 한 화면에 공존해야 이 스타일이다.

■ 접근성·안전장치
- prefers-reduced-motion: 그리드 전개·크롬 스윕·별 트윙클·글리치·스캔라인 밀림 전부 정지,
  최종 상태만 즉시 표시(글로우·크롬 그라디언트·색은 유지 — 스캔라인 오버레이는 정적으로 남김).
- JS 실패 폴백: 파티클/각도 회전이 JS 의존이면, CSS만으로 정적 크롬 텍스트 + 정적 원근 그리드 배경 +
  정적 스캔라인이 렌더되도록 기본값 보장(순수 CSS 그라디언트 그리드 + repeating-linear-gradient 스캔라인).
- focus-visible: 시안 #22E1FF 2px 아웃라인 + 2px offset — 네온 글로우와 구분되는 또렷한 포커스 링.
- 본문 가독성: 다크 배경 위 한글 본문은 #E7EEF7 이상, 표 숫자는 4.5:1 확보. 크롬 텍스트는 장식(헤더·KPI 라벨)에만, 본문엔 금지.
- 스캔라인 opacity는 .08 상한 — 본문 위를 지날 땐 가독성 우선으로 더 옅게.

■ 금지
차분한 파스텔, 미니멀 여백, 세리프, 절제된 무채색 팔레트, 얌전한 톤,
평평한 단색 카드(글로우·홀로그램 보더 없는), 라이트 화이트 배경, 원근 그리드 없는 히어로,
크롬·홀로그램·네온·스캔라인 중 하나라도 빠진 화면, 한글 본문에 미래틱 이탤릭 강제(가독성 파괴).
[클러스터 분리 — 아래는 옆 스타일처럼 보이게 만드는 요소라 절대 금지]
· 매트 무광 표면·불투명 플랫 원색 블록(빨강/검정/흰색 덩어리) — 브루탈리즘식 무광 평면. 이 스타일의 표면은 항상 금속 반사·홀로그램이어야 함.
· 각지고 무장식인 흑백 박스·굵은 검정 외곽선·직각 하드 엣지 골격 — 브루탈리즘식 구조. 크롬 브래킷·홀로그램 보더·radius 14px·네온 글로우를 반드시 유지.
· 시스템 산세리프 흑백 대비만으로 밀어붙인 레이아웃(글로우·그라디언트 부재) — 브루탈리즘식 타이포. 헤더는 반드시 크롬 그라디언트 클립.
· 형광 색 패턴 스티커·낙서·콜라주·랜덤 그래픽 도배(잡다한 이모지/도형 산개) — 맥시멀리즘식 장식 과잉. 여기서 "과잉"은 오직 크롬·홀로그램·네온 격자의 겹침이지, 패턴 스티커 도배가 아님.
· 원색 형광 배경 위 대비 패턴(체크·물방울·지그재그) 타일링 — 맥시멀리즘식. 배경은 항상 다크 퍼플 심야 그라디언트 + 원근 격자 + 스캔라인.

■ 자체 크리틱 루프
1) 크롬 그라디언트 텍스트 · 신스웨이브 원근 그리드 · 스캔라인 오버레이 · 홀로그램 보더가 넷 다 화면에 있는가? → 하나라도 없으면 시그니처 미완성.
2) 크롬 텍스트의 금속 반사가 하이라이트-미드-하이라이트로 두 번 이상 교차하는가, 아니면 밋밋한 회색 단일 그라디언트인가? → 단조 = 크롬 실패.
3) 마젠타·시안·퍼플·라임시안 홀로그램 색이 실제로 순환·공존하는가, 아니면 한 색만 남았는가? → 단색화 = 회귀.
4) 표면이 무광 플랫(브루탈리즘식)이거나 패턴 스티커로 도배(맥시멀리즘식)돼 옆 스타일처럼 보이지 않는가? → 그러면 즉시 크롬·홀로그램·격자로 되돌림.
5) 다크 배경 위 한글 본문/표 숫자가 4.5:1을 넘는가? → 미달 지점 색 보정.
6) reduced-motion·JS 실패 시에도 크롬 텍스트·원근 그리드·스캔라인이 정적으로 살아있는가? → 폴백 전수 검사.
```
