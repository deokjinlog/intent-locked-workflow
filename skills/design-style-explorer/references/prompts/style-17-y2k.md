# STYLE 17 — Y2K 베이퍼웨이브 (매력 축: 복고미래/과잉)

매력 가설: 2000년대 웹의 촌스러운 미래 상상이 지금 보면 오히려 짜릿하다.
매력은 절제가 아니라 과잉 — 크롬·홀로그램·네온 그리드를 겹쳐 쌓는 뻔뻔함이 정체성이다. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — Y2K 베이퍼웨이브 v2]

■ 무드
밀레니엄 전환기 웹 + 신스웨이브 나이트드라이브. 크롬 메탈로 각인된 미래,
자정의 퍼플 하늘 아래 무한히 뻗는 네온 격자. 세련보다 강렬. 얌전한 순간 실패.

■ 팔레트
배경 그라디언트: 상단 #1A0B2E → 중단 #2D0B45 → 하단 #0B0221 (다크 퍼플 심야) /
홀로그램 축: 마젠타 #FF2FB9 ↔ 시안 #22E1FF ↔ 퍼플 #A24BFF (세 색을 그라디언트로 순환) /
네온 하이라이트 핑크 #FF6EC7 / 그리드 라인 #FF2FB9~#22E1FF 그라디언트 /
크롬 텍스트 그라디언트: #EFF6FF → #9FB4CC → #FFFFFF → #6B7A99 → #E7EEF7 (금속 반사 5스톱) /
문제/경고 #FF3B6B / 성공 #3BFFB0
색은 겹칠수록 좋다 — 무채색 절제는 이 스타일의 죽음.

■ 타이포
디스플레이(헤더·KPI 숫자·차트 라벨): Orbitron 700~900 또는 Chakra Petch 600, italic 살짝 기울임(-6deg skew 또는 font-style italic).
한글 본문·표: Pretendard 400~600 (미래틱 폰트는 라틴/숫자 전용, 한글은 가독성 우선).
헤더 텍스트는 letter-spacing 0.08em + uppercase(라틴) — 디지털 전광판 감각.

■ 형태
카드: 반투명 다크 퍼플 #2D0B45 at 60% + 1px 홀로그램 그라디언트 보더(마젠타→시안).
radius 14px, 외곽 네온 글로우 box-shadow: 0 0 24px rgba(255,47,185,.35).
모서리에 작은 크롬 브래킷(⌐ ⌐ 꺾쇠) 장식 허용. 배경엔 은은한 스캔라인 오버레이(반복 2px 가로줄, opacity .05).

■ 히어로 연출 — 로드 시퀀스
0.0s: 화면 암전(#0B0221)에서 신스웨이브 그리드가 지평선(수직 60% 지점)에서 아래로 원근 전개 — perspective 격자가 800ms에 걸쳐 화면 하단까지 깔린다.
0.3s: 그리드 위로 크롬 타이틀이 스캔되듯 위→아래 리빌(clip-path 또는 mask 스윕, 600ms) + 금속 그라디언트 하이라이트가 좌→우로 1회 훑고 지나감.
0.9s: KPI 4장이 시차 120ms 간격으로 아래에서 튀어오르며 네온 글로우 점등(각 400ms).
1.5s: 우상단에 별/반짝임 파티클(✦) 4~6개가 랜덤 위치에서 트윙클 시작(무한 루프).

■ 마이크로 인터랙션
1) 카드 hover: 홀로그램 보더 그라디언트가 각도 회전(--angle 0→360deg, 1.2s), 글로우 반경 +8px 확대.
2) KPI 숫자 hover: 크롬 그라디언트 하이라이트 스윕이 좌→우로 재생(500ms) — 금속에 빛이 미끄러지는 감각.
3) 차트 막대/라인 hover: 마젠타↔시안 글로우 펄스(700ms) + 값 툴팁이 네온 보더로 팝.
4) 테이블 행 hover: 좌측 2px 홀로그램 바 점등 + 행 배경 rgba(162,75,255,.12).
5) 버튼 active: 크롬 표면이 반전(그라디언트 방향 뒤집힘) + 짧은 글리치 시프트(2px, 80ms).

■ 시그니처
크롬 그라디언트 텍스트(금속 반사 5스톱, background-clip:text)로 찍힌 타이틀 +
화면 하단을 가로지르는 신스웨이브 원근 그리드 지평선(마젠타→시안 라인, 아래로 소실).
이 둘이 한 화면에 공존해야 이 스타일이다.

■ 접근성·안전장치
- prefers-reduced-motion: 그리드 전개·크롬 스윕·별 트윙클·글리치 전부 정지, 최종 상태만 즉시 표시(글로우·색은 유지).
- JS 실패 폴백: 파티클/각도 회전이 JS 의존이면, CSS만으로 정적 크롬 텍스트 + 정적 그리드 배경이 렌더되도록 기본값 보장(순수 CSS 그라디언트 그리드).
- focus-visible: 시안 #22E1FF 2px 아웃라인 + 2px offset — 네온 글로우와 구분되는 또렷한 포커스 링.
- 본문 가독성: 다크 배경 위 한글 본문은 #E7EEF7 이상, 표 숫자는 4.5:1 확보. 크롬 텍스트는 장식(헤더)에만, 본문엔 금지.

■ 금지
차분한 파스텔, 미니멀 여백, 세리프, 절제된 무채색 팔레트, 얌전한 톤,
평평한 단색 카드(글로우·보더 없는), 라이트 화이트 배경, 단정한 그리드 없는 히어로,
크롬/홀로그램/네온 중 하나라도 빠진 화면, 한글 본문에 미래틱 이탤릭 강제(가독성 파괴).

■ 자체 크리틱 루프
1) 크롬 그라디언트 텍스트와 신스웨이브 그리드가 둘 다 화면에 있는가? → 없으면 시그니처 미완성.
2) 마젠타·시안·퍼플 세 홀로그램 색이 실제로 순환·공존하는가, 아니면 한 색만 남았는가? → 단색화 = 회귀.
3) 다크 배경 위 한글 본문/표 숫자가 4.5:1을 넘는가? → 미달 지점 색 보정.
4) reduced-motion·JS 실패 시에도 크롬 텍스트와 그리드가 정적으로 살아있는가? → 폴백 전수 검사.
```
