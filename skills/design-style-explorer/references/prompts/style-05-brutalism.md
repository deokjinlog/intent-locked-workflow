# STYLE 05 — 브루탈리즘 (매력 축: 대담함)

매력 가설: 뻔뻔한 자신감. 규칙을 아는 사람이 규칙을 깬다.
고도화 = 타이포의 폭력적 스케일 + 물리적 반응. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 브루탈리즘 v2]

■ 무드
웹 브루탈리즘. 포스터가 화면이 된 것처럼 — 멀리서도 읽히는 대담함.

■ 팔레트
배경 #FFFFFF / 잉크 #000000 / 원색 #FF3B00, #0037FF, #FFE600 — 3색 이내.
원색은 점이 아니라 "면"으로 과감하게 쓴다 (섹션 배경, 상태 블록).

■ 타이포
제목·수치: Archivo Black (또는 최고 굵기 산세리프). 본문 Pretendard.
KPI 대표 수치 1개는 화면에서 가장 큰 요소(72px 이상) — 숫자가 곧 레이아웃이다.

■ 형태·질감
3~4px 검정 실선 보더, radius 0, 오프셋 하드 섀도(box-shadow: 8px 8px 0 #000),
일부 요소 미세 회전(-1~1deg), 원색 면 분할 배경.

■ 히어로 연출 — 등장 대신 충격
애니메이션 오프닝 없음. 대신 첫 화면의 구도로 승부:
가장 중요한 KPI 하나를 초대형으로 상단에, 나머지 지표가 그 아래 정렬.
보는 순간 "숫자 하나가 화면을 지배"해야 한다.

■ 마이크로 인터랙션 — 물리성
1) 카드/버튼 hover: 4px 이동하며 섀도가 8px→4px로 줄어듦 — "찍히는" 반응
2) active(클릭): 완전히 붙음 (이동 8px, 섀도 0)
3) 테이블 행 hover: 배경이 옐로(#FFE600)로 반전
모든 반응은 100ms 이내 즉각적으로 — 브루탈리즘은 뜸들이지 않는다.

■ 시그니처
찍히는 hover + 초대형 숫자.

■ 접근성·안전장치
- prefers-reduced-motion: hover 이동 제거, 색 변화만 유지
- 회전된 요소 안의 본문 텍스트는 수평 유지 (가독성)

■ 금지
그라디언트, 블러, 부드러운 그림자, 파스텔, 1px 얇은 보더, 라운드,
소심한 크기, 4색 이상의 원색

■ 자체 크리틱 루프
1) 3m 떨어져서 봐도 포스터처럼 읽히는가?
2) 원색이 4색 이상인가? → 줄여라
3) "무난한" 요소가 있는가? → 키우거나 기울여라
4) 대담함이 가독성을 죽였는가? → 대담함≠난독. 정보는 읽혀야 한다
```
