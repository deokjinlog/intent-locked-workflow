# STYLE 05 — 브루탈리즘 (매력 축: 구조적 대담)

매력 가설: 뻔뻔한 자신감. 규칙을 아는 사람이 규칙을 깬다.
고도화 = 무채색 규율 + 단 하나의 원색 + 타이포의 폭력적 스케일 + 물리적 반응.
색으로 꾸미지 않는다. 구조와 크기와 검정 보더로 압도한다. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 브루탈리즘 v2]

■ 무드
웹 브루탈리즘. 포스터가 화면이 된 것처럼 — 멀리서도 읽히는 대담함.
색이 아니라 "구조"로 압도한다. 흑·백·회색의 규율 위에 원색 한 방울.
raw HTML 이 그대로 드러난 듯한 골격 — 다듬지 않은 각진 블록의 힘.

■ 팔레트 — 무채색 + 원색 딱 1개
배경 #FFFFFF / 잉크 #000000 / 보조 회색 2단 #E7E7E7, #1A1A1A.
원색은 오직 하나: #FF3B00 (블레이즈 오렌지레드). 다른 원색은 존재하지 않는다.
이 1색은 점이 아니라 "면"으로 딱 1~2곳에만 (지배 KPI 블록 배경 또는 단일 강조 바).
화면의 90% 이상은 흑·백·회색. 원색이 흔해지면 힘을 잃는다 — 아껴서 때린다.

■ 타이포 — 숫자가 곧 레이아웃
제목·수치: Archivo Black (또는 최고 굵기 산세리프, font-weight 900). 본문 Pretendard.
제목 스케일 clamp(48px, 10vw, 160px). letter-spacing -0.03em 로 글자를 서로 밀착.
KPI 대표 수치 1개는 화면에서 가장 큰 요소 — 최소 120px, 가능하면 160px 이상.
숫자 하나가 한 화면을 통째로 차지해도 좋다. text-transform: uppercase, 라벨은 볼드 소문자 대비.

■ 형태·질감 — 두꺼운 검정 보더 + 하드 섀도
4~6px 검정 실선 보더 (얇은 선 금지), radius 0 절대 준수 (모서리는 무조건 각).
오프셋 하드 섀도 box-shadow: 10px 10px 0 #000 — 흐림값 0, 항상 검정 단색.
일부 요소 미세 회전 -1.5~1.5deg 로 정렬을 일부러 어긋냄.
질감·패턴·텍스처 배경 없음 — 면은 순수 단색(흰/회/검)으로만 채운다.

■ 레이아웃 — 비대칭이 기본값
격자를 일부러 깬다. 컬럼 폭을 불균등하게 (예: 2fr / 1fr / 1.4fr), 블록마다 오프셋.
가운데 정렬·좌우 대칭 회피 — 무게를 한쪽으로 몰고 여백을 반대편에 크게 남긴다.
블록끼리 검정 보더가 맞닿거나 살짝 겹치게 배치해 "조립된 골격" 느낌을 낸다.

■ 히어로 연출 — 등장 대신 충격
애니메이션 오프닝 없음. 대신 첫 화면의 구도로 승부:
가장 중요한 KPI 하나를 초대형으로, 원색 면 위에 얹어 상단 한쪽에 몰아친다.
나머지 지표는 그 아래 흑백 블록으로 비대칭 정렬.
보는 순간 "숫자 하나 + 원색 면 하나"가 화면을 지배해야 한다.

■ 마이크로 인터랙션 — 물리성
1) 카드/버튼 hover: 4px 이동하며 섀도가 10px→4px로 줄어듦 — "찍히는" 반응
2) active(클릭): 완전히 붙음 (이동 10px, 섀도 0)
3) 테이블 행 hover: 배경이 검정(#000)으로 반전, 텍스트 흰색 — 원색 아님, 흑백 규율 유지
모든 반응은 100ms 이내 즉각적으로 — 브루탈리즘은 뜸들이지 않는다.

■ 시그니처
찍히는 하드 섀도 hover + 초대형 숫자 + 원색 단 1색의 면.

■ 접근성·안전장치
- prefers-reduced-motion: hover 이동 제거, 섀도 축소만 유지 (색 변화 없음)
- 회전된 요소 안의 본문 텍스트는 수평 유지 (가독성)
- 원색 #FF3B00 면 위 텍스트는 흰색 또는 검정으로 대비 4.5:1 이상 확보

■ 금지
- 원색 2색 이상 (다색 팔레트) — 무채색+1색 규율을 절대 깨지 않는다
- 형광 다색·비비드 조합 (19 맥시멀리즘처럼 보이게 하는 요소)
- 패턴/텍스처 배경 (도트·스트라이프·격자·노이즈·체커) — 면은 단색만
- 장식 도형 (별·물결·blob·스티커·이모지 뱃지·리본)
- 그라디언트 (선형·방사형·conic 전부)
- 크롬·홀로그램·메탈릭·글로시·광택·반짝임 일체 (17 Y2K처럼 보이게 하는 요소)
- 블러, 부드러운/컬러 그림자, 파스텔, 1px 얇은 보더, 라운드 모서리
- 소심한 크기, 대칭 중앙 정렬, "무난한" 정돈된 그리드

■ 자체 크리틱 루프
1) 3m 떨어져서 봐도 포스터처럼 읽히는가?
2) 화면에 원색이 2색 이상 보이는가? → 1색으로 줄여라
3) 원색이 화면 10% 넘게 쓰였는가? → 면을 줄이고 흑백 비중을 키워라
4) 그라디언트·패턴·광택·장식 도형이 하나라도 있는가? → 전부 제거 (클러스터메이트化)
5) 레이아웃이 대칭·정돈됐는가? → 컬럼을 어긋내고 무게를 한쪽으로 몰아라
6) "무난한" 요소가 있는가? → 키우거나 기울이거나 검정 보더로 감싸라
7) 대담함이 가독성을 죽였는가? → 대담함≠난독. 정보는 읽혀야 한다
```
