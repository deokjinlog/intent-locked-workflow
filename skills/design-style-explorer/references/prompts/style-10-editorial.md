# STYLE 10 — 에디토리얼 매거진 (매력 축: 지성)

매력 가설: 읽는 맛. 데이터에 서사를 입히면 기억된다. 고도화 = 모션이 아니라 편집.
마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 에디토리얼 매거진 v2]

■ 무드
종이 경제지의 데이터 저널리즘 지면. 대시보드를 "읽는다".

■ 팔레트
종이 #FAF7F2 / 잉크 #1C1B18 / 진홍 #A32638 / 헤어라인 #D8D2C6

■ 타이포
제목: Noto Serif KR 700, 큰 크기 / 수치도 세리프로 크게(44px 이상) /
본문: Pretendard / 캡션: 세리프 이탤릭 느낌의 작은 크기

■ 형태·질감
신문식 컬럼 그리드, 섹션 상단 2px+1px 이중 괘선, 내부 1px 헤어라인,
radius 0, 리드 문단에 드롭캡 1곳. 괘선은 수평 위주 — 수직 괘선 남발 금지.

■ 히어로 연출 — 편집이 곧 연출
모션은 전체 일괄 페이드인(400ms) 하나만.
대신 첫인상은 편집으로: 최상단에 신문 제호처럼 [제품명 · 날짜 · 판번호],
그 아래 오늘의 데이터를 요약한 리드 문장 1줄 (세리프, 헤드라인 톤).

■ 고도화 포인트 — 데이터 저널리즘 (전부 필수)
1) 각 KPI 아래 헤드라인 톤 한 줄 해설 (예: "지출, 전월 대비 8% 꺾여")
   — 해설은 반드시 실제 데이터와 일치할 것. 서사≠과장
2) 차트에 기사 캡션 ("4주차 84만원, 기간 내 최고치")
3) 표는 신문 시세표 문법 — 촘촘하되 괘선은 수평만, 숫자 우측 정렬

■ 마이크로 인터랙션
1) 테이블 행 hover: 형광펜 밑줄 느낌의 옅은 배경
2) 링크: 진홍색 + 밑줄
그 이상의 반응 금지 — 종이는 움직이지 않는다.

■ 시그니처
숫자마다 붙는 헤드라인 한 줄 — 데이터가 기사가 된다.

■ 접근성·안전장치
- prefers-reduced-motion: 페이드인 제거
- 세리프 수치에도 tabular-nums 적용 (font-variant-numeric)

■ 금지
카드 그림자, 알약 태그, 네온, 다크 배경, 스태거 모션, 이모지,
수직 괘선 남발, 데이터와 어긋나는 해설 문구

■ 자체 크리틱 루프
1) 해설 문장이 실제 수치와 일치하는가? → 하나라도 다르면 수정
2) 지면처럼 위계(제호→리드→본문→각주)가 한눈에 읽히는가?
3) 세리프(제목·수치)와 산세리프(본문) 역할 구분이 일관되는가?
4) 여백이 종이답게 넉넉한가? 촘촘한 곳은 표뿐인가?
```
