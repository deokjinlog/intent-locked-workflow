# STYLE 07 — 미니멀 모노크롬 (매력 축: 정밀함)

매력 가설: 감탄은 "무엇을 뺐는데도 완벽한가"에서 온다.
고도화 = 효과가 아니라 수학. 무채색 세계에 주황이 정확히 3번 켜질 때, 그 3번이 화면 전체를 지배한다. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 미니멀 모노크롬 v2]

■ 무드
"모노크롬 절제." 잉크와 종이, 그리고 단 하나의 주황.
스위스적 규율을 빌리되 스위스 포스터가 아니다 — 대형 유채 타이포로 소리치는 격자 포스터(클러스터메이트 18)의 정반대편이다.
여기선 침묵이 매체다. 90%가 흰 여백, 9%가 무채색 잉크, 1%가 주황. 그 1%가 유일하게 말한다.
공공 포털의 "파란 표로 얻는 신뢰"(클러스터메이트 02)와도 다르다 — 신뢰는 괘선이 아니라 여백과 정렬의 완벽함에서 온다.

■ 팔레트 — 무채색 5단 + 주황 1
배경 #FFFFFF / 잉크 #111111 / 중간 회색 #767676 / 연회색 #A8A8A8 / 헤어라인 #E5E5E5 / 포인트 #FF4D00
무채색은 순수 그레이스케일만 — R=G=B 강제. 회색에 파란기(#7A8290류)·붉은기(#8A7676류) 절대 금지. 차가움도 따뜻함도 없는 순수 중성.
포인트 #FF4D00 사용처는 정확히 3곳, 그 이상도 이하도 아니다:
  ① 차트에서 가장 중요한 단 하나의 피크(막대·점·라인 꺾임 중 하나)
  ② "문제" 상태를 알리는 단 하나의 인디케이터(2px 세로 바 또는 6px 점)
  ③ ②와 연결된 KPI 수치 단 하나
그 외 100% 무채색. 주황은 배경·큰 면·테두리·아이콘에 절대 쓰지 않는다 — 오직 잉크 획처럼 작고 날카롭게.

■ 타이포 — 유일한 장식이므로 수학적으로
Pretendard 단일 패밀리. 크기 5단계 고정: 11 / 16 / 24 / 36 / 54px (1.5배 기하급수) — 이 외 어떤 값도 금지.
KPI 수치 54px 굵기 800 자간 -0.04em / 섹션 제목 24px 굵기 700 / 본문 16px 굵기 400 / 레이블 11px 굵기 500 자간 .08em 대문자화.
크기비 최대:최소 = 54:11 ≈ 5배. 이 5배 대비가 색을 대신한다.
정렬은 좌측 정렬 단일 축 — 초대형 타이포를 캔버스 대각선에 비스듬히 앉히거나 화면 밖으로 흘리는 비대칭 연출(클러스터메이트 18) 금지. 모든 텍스트는 8px 그리드 위에 반듯이 눕는다.
행간 1.5 고정. 폭 넓은 제목은 max-width 로 60ch 이내 통제 — 타이포가 레이아웃을 지배하지 않고 봉사한다.

■ 그리드 — 정밀함의 실체이되 보이지 않는다
8px 베이스라인 그리드에 전 요소를 정렬하되, 그리드 선 자체는 절대 렌더링하지 않는다. 격자는 느껴질 뿐 보이지 않는다(클러스터메이트 18의 노출된 굵은 괘선 격자와 정반대).
여백 4단계(8/16/32/64px)만 사용. 섹션 간 수직 리듬은 64px 고정 — 이 극단적 공백이 시그니처다.
카드·박스·배경색 면·둥근 컨테이너 전부 금지. 구획은 1px 헤어라인(#E5E5E5)과 64px 여백만으로.
KPI 4개는 세로 헤어라인으로만 분리 — 상자에 넣지 마라. 표를 그릴 땐 가로 헤어라인은 헤더 아래 1줄과 마지막 행 아래 1줄, 딱 2줄만. 셀마다 괘선을 두르는 격자형 표(클러스터메이트 02) 금지 — 정렬만으로 열을 읽게 한다.

■ 히어로 연출 — 거의 없음이 연출이다
허용 모션 단 하나: 진입 시 전체 400ms 일괄 페이드인(opacity 0→1). translateY·scale 동반 금지.
카운트업·차오름·바운스·블롭·오로라·글로우·스태거·패럴랙스 전부 금지. 정지 화면의 완성도로 승부.
로딩 스켈레톤조차 회색 펄스 없이 정적 헤어라인으로.

■ 마이크로 인터랙션 (2~3개 한정 — 절제가 규칙)
1) 차트 막대 hover: 해당 막대만 잉크색(#111111) 유지, 나머지 전 막대가 #E5E5E5로 물러난다
   (강조가 아니라 "후퇴에 의한 강조" — 밝히지 않고 주변을 죽여서 도드라지게 한다)
2) 테이블 행 hover: 배경색 변화 없이 좌측 2px 잉크 바만 나타난다
3) (선택) KPI hover: 수치 아래 1px 헤어라인이 좌→우로 200ms에 그어진다
전이는 전부 150~200ms ease. 색이 바뀌는 hover(파란 하이라이트·붉은 강조) 금지 — 무채색의 명도만 움직인다.

■ 시그니처
후퇴에 의한 강조(주변을 #E5E5E5로 죽여 대상을 세운다) + 무채색 바다에 정확히 3곳에만 켜지는 #FF4D00 + 64px 극단 여백.

■ 접근성·안전장치
- prefers-reduced-motion: 페이드인 제거, 즉시 표시
- 회색 텍스트(#767676)는 12px 이상에서만, 연회색(#A8A8A8)은 비텍스트 헤어라인·구분선에만
- 주황 #FF4D00은 흰 배경 대비 명도가 낮으니 11px 미만 텍스트에 쓰지 않는다 — 수치는 크게

■ 금지 (클러스터메이트와 벌리는 항목 = 핵심)
- 파란기/붉은기 도는 회색, 두 번째 유채색 — 주황 외 모든 색상(무채색 순수 그레이스케일만)
- 파랑·빨강 주액센트 계열 전면 금지 — 파란 표 신뢰 연출(02), 빨간 격자 강조(18) 흉내 금지
- 셀마다 괘선을 두른 격자형 표·괘선 남발 — 표는 헤어라인 2줄 + 정렬만(02의 표 문법 금지)
- 노출된 굵은 그리드 선·눈에 보이는 격자·컬럼 구분 세로 굵은 선(18의 격자 노출 금지)
- 비대칭 초대형 타이포, 대각선/화면밖으로 흐르는 제목, 캔버스를 압도하는 헤드라인(18의 타이포 연출 금지)
- 유채색 대형 타이포(주황·파랑·빨강으로 채운 거대 제목) 금지 — 큰 글자는 잉크색 수치뿐
- 그림자, 카드 박스, 배경색 면, 둥근 컨테이너(radius 0 고정)
- 아이콘·이모지·일러스트·픽토그램, 그라디언트, 글로우, 스태거, 패럴랙스
- 정의 외 폰트 크기(11/16/24/36/54px)·여백(8/16/32/64px) 일체

■ 자체 크리틱 루프
1) 요소를 하나 더 뺄 수 있는가? → 뺄 수 있으면 빼라. 뺄 게 없을 때가 완성이다
2) 모든 크기·여백이 정의된 스케일(11/16/24/36/54 · 8/16/32/64) 위에 있는가? → 하나라도 어긋나면 수정
3) 주황이 정확히 3곳인가? → 4곳이면 실패(하나 지워라), 2곳이면 재배치(하나 살려라). 3이 아니면 불합격
4) 헤어라인을 전부 지워도 구조가 읽히는가? → 읽히면 합격(여백과 정렬이 제 역할을 한다는 증거)
5) 이 화면을 흑백 프린트하면 02·18과 구분되는가? → 색 없이도 "괘선 없음 + 격자 안 보임 + 좌측 정렬 절제"로 즉시 07임이 드러나야 한다. 아니면 형태를 더 벌려라
```
