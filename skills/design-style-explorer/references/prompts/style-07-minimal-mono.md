# STYLE 07 — 미니멀 모노크롬 (매력 축: 정밀함)

매력 가설: 감탄은 "무엇을 뺐는데도 완벽한가"에서 온다.
고도화 = 효과가 아니라 수학. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 미니멀 모노크롬 v2]

■ 무드
스위스 타이포그래피 포스터가 대시보드가 된다면. 잉크와 종이, 그리고 단 하나의 주황.

■ 팔레트
배경 #FFFFFF / 잉크 #111111 / 중간 회색 #767676 / 헤어라인 #E5E5E5 / 포인트 #FF4D00
포인트 사용처는 정확히 3곳: ①차트 피크 ②"문제" 상태 ③문제 관련 KPI 수치. 그 외 전부 무채색.

■ 타이포 — 유일한 장식이므로 수학적으로
Pretendard 단일. 크기 5단계 고정: 11 / 16 / 24 / 36 / 54px (1.5배 기하급수) — 이 외 금지
KPI 수치 54px 굵기 800 자간 -0.04em / 레이블 11px 굵기 500 자간 .08em
크기비 최대:최소 ≈ 5배. 이 대비가 색을 대신한다.

■ 그리드 — 정밀함의 실체
8px 베이스라인 그리드에 전 요소 정렬. 여백 4단계(8/16/32/64px)만 사용.
카드·박스·배경색 면 금지 — 구획은 1px 헤어라인과 64px 여백만으로.
KPI 4개는 세로 헤어라인으로만 분리. 상자에 넣지 마라.

■ 히어로 연출 — 거의 없음이 연출이다
허용 모션 단 하나: 전체 400ms 일괄 페이드인.
카운트업·차오름·바운스·블롭·스태거 전부 금지. 정지 화면의 완성도로 승부.

■ 마이크로 인터랙션 (2~3개 한정 — 절제가 규칙)
1) 차트 막대 hover: 해당 막대만 잉크색 유지, 나머지가 #E5E5E5로 물러난다
   (강조가 아니라 "후퇴에 의한 강조")
2) 테이블 행 hover: 배경 변화 없이 좌측 2px 잉크 바만
3) (선택) KPI hover: 수치 아래 1px 헤어라인이 좌→우로 그어진다

■ 시그니처
후퇴에 의한 강조 + 정확히 3곳에만 존재하는 주황.

■ 접근성·안전장치
- prefers-reduced-motion: 페이드인 제거
- 회색 텍스트(#767676)는 12px 이상에서만 사용

■ 금지
그림자, 카드 박스, 배경색 면, 아이콘·이모지, 라운드(radius 0 고정),
두 번째 유채색, 스태거, 그라디언트, 정의 외 폰트 크기·여백

■ 자체 크리틱 루프
1) 요소를 하나 더 뺄 수 있는가? → 뺄 수 있으면 빼라. 뺄 게 없을 때가 완성이다
2) 모든 크기·여백이 정의된 스케일 위에 있는가? → 하나라도 어긋나면 수정
3) 주황이 정확히 3곳인가? → 4곳이면 실패, 2곳이면 재배치
4) 헤어라인을 전부 지워도 구조가 읽히는가? → 읽히면 합격 (여백이 제 역할)
```
