# STYLE 19 — 맥시멀리즘 (매력 축: 과잉/폭발)

매력 가설: 절제가 아니라 폭발이 시선을 붙든다. 매력은 "더는 못 넣겠다" 직전의 밀도.
고도화 = 무질서처럼 보이되 실은 통제된 충돌. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 맥시멀리즘 v2]

■ 무드
안티 미니멀. 색·패턴·타이포가 동시에 터지는 브랜드 랜딩/포스터.
Memphis·anti-design 계보 — 화면이 조용할 틈을 주지 않는다.
"비어 있으면 채운다"가 규칙. 단, 채움에는 리듬이 있다.

■ 팔레트 — 다색 원색·형광 충돌
배경 #FFF8E7 크림 위에 도형 레이어 폭발 /
마젠타 #FF2E93 / 라임 #C6FF00 / 오렌지 #FF6B00 / 전기파랑 #0038FF /
보조 노랑 #FFD400 / 텍스트는 잉크블랙 #0A0A0A (본문 가독은 검정이 잡는다)
5색 이상 동시 사용 강제 — 한 화면에 4색 이하로 떨어지면 회귀.
색면은 서로 인접 배치해 대비 극대 (마젠타 옆 라임, 파랑 옆 오렌지).

■ 타이포 — 폰트 믹스 필수
헤드라인 Archivo Black 또는 Anton (초대형, 900) /
서브·라벨 Space Grotesk 500~700 (기하학적 대비) /
한글 본문 Pretendard 400~700.
최소 2개 디스플레이 폰트 혼용 — 크기 편차 극단 (h1 64px+ vs 라벨 12px).
헤드라인은 -3°~6° 회전, 글자별 색 다르게, 일부 아웃라인(text-stroke)만.

■ 형태 — 도형·패턴·겹침
배경에 도형 산포: 원·삼각형·지그재그·물결·별·체크무늬 SVG 레이어.
카드는 각기 다른 회전(-4°~4°) + 굵은 검정 보더 3~4px + 하드 오프셋 그림자
(box-shadow: 8px 8px 0 #0A0A0A) — 블러 0.
요소끼리 의도적으로 겹침(z-index 교차) — KPI 카드 모서리가 차트를 물고,
스티커 도형이 헤더 글자 위로 삐져나온다.

■ 히어로 연출 — 폭발 시퀀스 (로드 0~1.4s)
0.0s 크림 배경만.
0.1s 배경 도형들이 서로 다른 방향에서 튕겨 들어옴 (각 80~140ms 스태거, overshoot).
0.5s 초대형 헤드라인이 좌측에서 밀려들며 살짝 회전 오버슈트 후 정착.
0.8s KPI 4장이 팡팡 팝인 (scale 0.6→1.08→1, 각 60ms 간격).
1.1s 스티커/도형 남발 레이어가 마지막에 "탁" 찍히며 등장 (약한 회전 진동).
차트 막대는 형광색으로 아래에서 차오름 (600ms, cubic-bezier 튕김).

■ 마이크로 인터랙션 — 3개+ 번호로
1) 카드 hover: 하드 그림자 거리 8px→12px + 회전 각 반전(살짝 반대로 기울며 떠오름)
2) KPI 숫자 hover: 배경 색면이 다음 원색으로 스냅 교체 (150ms, 톤 튐)
3) 버튼/태그 active: 그림자가 0으로 붕괴하며 요소가 오른아래로 8px 눌림 (도장 찍기)
4) 스크롤 시 배경 도형 레이어가 서로 다른 속도로 패럴럭스 (앞 도형 빠르게, 뒤 느리게)
5) 차트 막대 hover: 형광 보색 테두리 점멸 + 값 말풍선이 회전된 스티커로 튀어나옴

■ 시그니처
색면 충돌 + 스티커/도형 남발 + 겹치는 레이어.
서로 다른 원색 블록이 경계선 없이 부딪히고, 그 위로 회전된 도형 스티커가
아무렇게나 붙은 듯 흩뿌려지며, 카드·차트·헤더가 z축으로 겹쳐 물린다.
이 세 가지가 한 화면에 동시에 없으면 맥시멀리즘이 아니다.

■ 접근성·안전장치
- prefers-reduced-motion: 폭발 등장·패럴럭스·점멸 전부 제거, 최종 정착 상태로 즉시 표시.
  누름(active) 반응만 유지.
- 본문 가독 사수: 텍스트는 반드시 크림/흰 색면 위 잉크블랙 배치 —
  형광 배경 위 본문 텍스트 금지 (형광은 도형·강조·색면 전용).
  KPI 숫자 등 형광 위 텍스트는 검정 + text-stroke 로 대비 확보.
- JS 실패 폴백: 애니메이션·패럴럭스가 안 돌아도 정적 레이아웃에서 색면 충돌과
  겹침·회전이 그대로 보이도록 CSS transform 초기값을 최종 상태로 설정.
- focus-visible: 형광 배경에 묻히지 않도록 검정 3px + 흰 1px 이중 아웃라인.
- 회전은 시각 장식만 — 실제 클릭 히트영역은 회전 전 사각으로 유지.

■ 금지 (정체성의 절반)
여백 위주 절제, 무채색·그레이스케일, 단일 폰트, 차분함·미니멀리즘,
얌전한 그리드 정렬(모든 카드가 각 없이 수평 정렬), 블러 있는 소프트 섀도,
파스텔 저채도, 4색 이하 팔레트, 겹침 없는 평면 배치, 회전 0°만 사용,
보더 없는 카드, 배경 단색·무패턴.

■ 자체 크리틱 루프
1) 한 화면에 원색/형광이 5색 이상 동시에 살아 있는가? → 4색 이하면 색 추가.
2) 색면 충돌·스티커 남발·겹치는 레이어 3요소가 모두 존재하는가? → 빠진 것 보강.
3) 정렬이 너무 얌전해졌는가(회전·오프셋 소실)? → 카드 각·z축 겹침 재도입.
4) 폭발 속에서도 본문 텍스트가 검정·고대비로 읽히는가(형광 위 본문 없는가)? → 색면 교정.
```
