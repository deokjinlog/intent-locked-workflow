# STYLE 12 — 뉴모피즘 소프트 (매력 축: 촉감)

매력 가설: 누르고 싶다. 매력은 시각이 아니라 촉각의 시뮬레이션.
고도화 = "눌림"의 물리학. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — 뉴모피즘 소프트 v2]

■ 무드
부드러운 실리콘 버튼의 물리 인터페이스. 화면에 실제 요철이 있는 것처럼.

■ 팔레트
배경 #E4E9F2 단일 — 카드도 같은 색 (뉴모피즘의 전제) /
텍스트 #3D4A5C / 포인트 #4A7CFF / 문제 #E36868
색 대비는 낮게 유지 — 대비는 색이 아니라 그림자가 만든다.

■ 타이포
Pretendard 600~700 — 흐린 배경 위 가독성 확보를 위해 굵게.

■ 형태 — 요철의 문법 (광원은 항상 좌상단)
볼록(raised): box-shadow: -8px -8px 16px #FFFFFF, 8px 8px 16px #C3CAD6
오목(inset):  같은 값의 inset. radius 20px.
역할 배분: KPI 카드·버튼 = 볼록 / 차트 트랙·테이블 영역·입력부 = 오목
— 요철 대비가 곧 위계다.

■ 히어로 연출 — 표면의 완성도
화려한 등장 없음. 로드 시 400ms 페이드인만.
차트 막대는 오목한 트랙 안에서 700ms 차오른다 (홈 안을 채우는 감각).

■ 마이크로 인터랙션 — 눌림 (이 스타일의 전부)
1) 카드/버튼 hover: 그림자 거리 -2px (살짝 떠오름)
2) active(클릭·터치): 그림자가 inset으로 반전되며 오목하게 눌린다 (150ms)
   — 이 요철 반전이 시그니처. 모든 인터랙티브 요소에 빠짐없이 적용
3) 차트 막대 hover: 포인트 블루 점등
4) 상태 태그: 작은 볼록 알약 (문제 상태만 오목 + 붉은 텍스트)

■ 시그니처
클릭하면 실제로 눌리는 표면 — 요철 반전.

■ 접근성·안전장치
- prefers-reduced-motion: 페이드인·차오름 제거 (눌림 반응은 접근성상 유지)
- 낮은 대비 보완: 본문 텍스트는 반드시 #3D4A5C 이상의 진한 색 (4.5:1 확보)
- focus-visible: 포인트 블루 2px 아웃라인 (그림자만으로는 포커스 식별 불가)

■ 금지
강한 색 대비, 보더 라인, 다크 배경, 플랫 카드, 배경과 다른 색의 카드,
하드 섀도, 순백 하이라이트 과다(회색빛 유지), 광원 방향 불일치

■ 자체 크리틱 루프
1) 광원이 전 요소에서 좌상단으로 일관되는가? → 그림자 방향 전수 검사
2) 볼록·오목 구분이 정보 위계와 일치하는가?
3) 대비 부족으로 텍스트가 묻힌 곳(4.5:1 미달)은? → 보정
4) 눌림 반응이 없는 인터랙티브 요소가 있는가? → 추가
```
