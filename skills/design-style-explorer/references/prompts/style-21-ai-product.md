# STYLE 21 — AI 프로덕트 (매력 축: 지능/미래)

매력 가설: 화면이 "생각하는 도구"처럼 느껴진다. 매력은 기능이 아니라 분위기 — 보라 그라디언트와 스파클, 은은한 글로우가 "이건 똑똑한 제품"이라는 인상을 인지 이전에 심는다.
고도화 = 그라디언트 액센트의 절제된 사용과 글로우의 미묘함. 남발하면 장난감, 아끼면 프로덕트. 마스터 프롬프트 뒤에 붙여 사용한다.

```
[스타일 지시 — AI 프로덕트 v2]

■ 무드
ChatGPT·Perplexity·Linear AI 의 감성. 조용하고 밝고 지적인 SaaS 콘솔.
배경은 거의 흰색인 라이트 톤, 액센트만 보라·인디고 그라디언트로 살아난다.
"AI 가 뒤에서 돕고 있다"는 미래감 — 요란함이 아니라 정제된 여백과 은은한 발광으로 표현.
차갑지 않고 부드럽다. 곡선, 무광 표면, 숨 쉬는 여백.

■ 팔레트 — 라이트 배경 + 보라·인디고 그라디언트 액센트
배경 #FAFAFB / surface #FFFFFF / surface-subtle #F4F4F7 / 보더 #E8E8EE
텍스트 강 #16161D / 텍스트 본문 #3A3A44 / 텍스트 약 #8A8A99
액센트 그라디언트: 보라 #7C5CFF → 인디고 #5B8DEF (135deg 기본)
액센트-강 #6A47F5 / 액센트-연 #EDE9FF (칩·배지 배경) / on-액센트 #FFFFFF
스파클·글로우 광원색 #9B7BFF (보라 계열, 알파 낮게)
상태: 성공 #12B76A / 경고 #F79009 / 문제 #F04438 (원색 아닌 살짝 톤다운)
그라디언트는 액센트에만 — 배경·본문·표면은 무채색 유지가 원칙.

■ 타이포 — Inter 모던
Inter (구글폰트) 명시 로드. 숫자는 tabular-nums, 한글 본문은 Pretendard 가정.
Display 34/42 700 · H1 26/34 700 · H2 20/28 600 · Title 16/24 600 ·
Body 14/22 400 · Caption 13/18 400 · Label 12/16 500 (배지·태그, letter-spacing 0.2px).
큰 제목은 -0.02em 트래킹으로 살짝 조여 모던하게. 본문은 기본 트래킹.
KPI 숫자는 28~34px 700, 그라디언트 텍스트 허용(액센트 숫자 1~2개 한정).

■ 형태
radius 넉넉 — 카드 16px, 버튼·인풋 10px, 칩·배지 pill(999px), 모달 20px, 아바타 원형.
보더 1px #E8E8EE 무광, 순검정 보더 금지. 카드 배경은 순백 surface.
글로우 카드: 카드 뒤에 보라 라디얼 글로우 은은하게(box-shadow 0 0 40px rgba(155,123,255,.10)
+ 0 1px 2px rgba(16,16,29,.04)). hover 시 글로우 반경·투명도만 살짝 증가.
아이콘 24px 스트로크 1.5, 액센트 아이콘은 그라디언트 stroke 허용.

■ 히어로 연출 — 지능이 깨어나는 순서 (로드 시퀀스)
0.0s 상단바가 위에서 페이드+y-6px→0 (200ms), 좌측 로고 옆 스파클 아이콘이 마지막에 반짝 1회
0.15s KPI 카드 4장이 왼→오 50ms stagger, y+10px→0 + 뒤 글로우가 0→규정 투명도로 서서히 켜짐 (각 260ms)
0.45s 차트 카드가 올라오며 라인이 좌→우로 그려짐(stroke-dashoffset 600ms ease-out),
      마지막 데이터 포인트에 글로우 도트가 톡 켜짐
0.6s "AI 인사이트" 배지가 스파클과 함께 스케일 0.8→1 로 등장, 문장이 타이핑되듯 좌→우 마스크 리빌 (500ms)
0.9s 테이블 행이 상단부터 30ms 간격 페이드인, 사이드바 활성 항목에 그라디언트 인디케이터 바가 슬라이드로 도착

■ 마이크로 인터랙션 (이 스타일의 실체)
1) 스파클 반짝: AI 관련 요소(인사이트 배지·생성 버튼) 옆 4-포인트 스파클이 3~5초마다
   1회 scale 1→1.2→1 + opacity 펄스(600ms). hover 시 즉시 1회 반짝.
2) 글로우 확장: 카드·버튼 hover 시 보라 라디얼 글로우 반경 40→52px + 투명도 .10→.16 (200ms).
   그림자 위치는 고정, 오직 발광만 숨쉰다.
3) 그라디언트 셔플: 주요 CTA(예: "AI 리포트 생성") hover 시 그라디언트 각도가 135°→115° 로
   부드럽게 회전(300ms) — 빛이 흐르는 느낌.
4) 인디케이터 슬라이드: 사이드바 항목 전환 시 좌측 3px 그라디언트 바가 다음 항목으로 슬라이드(220ms).
5) 스켈레톤 시머: 데이터 로딩 자리에 보라빛 시머가 좌→우로 흐르는 shimmer(1.4s loop) — "생성 중" 은유.
6) 포커스 링: 인풋·버튼 focus 시 액센트-연(#EDE9FF) 4px 글로우 링이 스르륵 확장(150ms).

■ 시그니처
보라→인디고 그라디언트 액센트 + 4-포인트 AI 스파클 아이콘 + 카드 뒤 은은한 보라 글로우.
이 셋이 겹쳐 화면을 즉시 "AI 프로덕트"로 읽히게 하는 각인. 특히 KPI/CTA/인사이트 배지에
그라디언트를 아껴 배치하고 나머지는 무채색으로 비워 두는 대비가 핵심.

■ 접근성·안전장치
- prefers-reduced-motion: 스파클 펄스·시머·타이핑 리빌·그라디언트 셔플 전부 정지,
  글로우는 고정 투명도로 즉시 표시(발광 애니메이션만 제거).
- JS 실패 폴백: 스파클은 정적 아이콘으로, 글로우는 CSS box-shadow 로 항상 렌더,
  타이핑 리빌 실패 시 문장 전체 즉시 노출.
- 대비: 그라디언트 텍스트는 KPI 숫자·대제목 등 큰 글자에만, 본문은 #3A3A44 무채색 4.5:1 확보.
  액센트 위 텍스트는 on-액센트 #FFFFFF 로 대비 유지, 그라디언트 위 회색 금지.
- focus-visible: 액센트-연 4px 글로우 링 + 1px 액센트-강 아웃라인(발광만으로 포커스 식별 금지).
- 글로우 절제: 한 화면 글로우 카드 남발 금지 — 위계 상위 카드에만, 나머지는 평평한 보더 surface.

■ 금지 (정체성의 절반)
강제 다크 배경(밝은 #FAFAFB 기본 — 다크는 선택 토글일 뿐 강제 X), 딱딱한 순수 원색
(#FF0000·#0000FF 등 톤다운 안 된 채도), 레트로·픽셀·8bit·도트 폰트,
밀집 고밀도 레이아웃(여백을 지운 빽빽한 그리드 — 이 스타일은 숨쉬는 여백이 생명),
순검정 1px 하드엣지 보더·브루탈리즘식 각진 박스, 그라디언트를 배경·본문·표면 전면에 도배
(액센트 한정 원칙 위반), 네온 사이버펑크식 과발광(은은한 글로우지 발광 폭탄 아님),
세리프 본문, radius 0 또는 4px 이하 각진 카드.

■ 자체 크리틱 루프
1) 그라디언트가 액센트(KPI/CTA/인사이트/인디케이터)에만 쓰였는가 — 배경·본문·표면에 새어나간 그라디언트 색출.
2) 스파클과 글로우가 "은은"의 선을 지켰는가 — 글로우 투명도 .20 초과·스파클 상시 점멸 등 과잉 발광 검사.
3) 배경이 라이트(#FAFAFB) 기본이며 여백이 충분히 숨쉬는가 — 다크 강제·밀집 밀도 이탈 검사.
4) 원색·픽셀·세리프·하드엣지 등 타 스타일 잔재가 섞이지 않았는가 — 팔레트·형태 이탈 색출.
```
