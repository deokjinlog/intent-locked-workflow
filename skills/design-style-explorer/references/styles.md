# 12 스타일 전체 정의 (design-style-explorer 참조)

시안 생성 전 **해당 스타일 블록을 반드시 읽고**, 매력 축·연출·시그니처·금지·크리틱 루프를 **전부** 반영한다.
이 MD 가 원본(source of truth) — 여기서 HTML 시안을 뽑는다. HTML 은 사람이 보는 결과물, 이 정의는 AI 작업 층.

## 공통 규칙

- **매력 축 = 이 스타일이 왜 매력적인가.** 연출·인터랙션은 매력 축을 강화하는 방향으로만.
- **안티 이식**: 매력 축에 안 맞는 기법 금지. 아래는 대표 오이식 —

| 스타일 | 이식 금지 | 이유 |
|---|---|---|
| 공공포털 | 카운트업·바운스·글로우 | 관공서 신뢰감 파괴 |
| 럭셔리 다크 | 바운스·빠른/튀는 모션 | 품격 파괴 |
| 미니멀 모노크롬 | 블롭·컬러 모션·그림자 | 뺄셈의 미학 파괴 |
| 브루탈리즘 | 부드러운 이징·스프링 | 투박함의 의도 파괴 |
| 레트로 픽셀 | 부드러운 보간·안티앨리어싱 곡선 | 8비트 각짐 파괴 |

- 연출 강도: HUD/글래스/레트로/트렌디 = 강해도 됨 / 공공·미니멀·에디토리얼·럭셔리 = 절제.

---

## STYLE 01 — SF 게임 HUD
- **매력 축**: 발광하는 정보 밀도. 어둠 속 전술 인터페이스가 주는 "지금 뭔가 통제되고 있다"는 긴장감.
- **팔레트**: 배경 #050810 / 시안 #00E5FF(주) / 마젠타 #FF2D78(경고·피크) / 앰버 #FFB62E(대기) / 텍스트 #C9EEF7
- **타이포**: 제목 Orbitron(대문자, 자간 넓게) / 수치 Share Tech Mono / 본문 Pretendard
- **연출**: 로드 시 스캔라인이 위→아래로 훑으며 패널 순차 점등, 수치는 짧은 글리치 후 확정(틱), 보더가 켜지듯 발광. (화려함이 본질 — 강하게 OK)
- **시그니처**: "지연/문제" 상태가 경고등처럼 깜빡임 + 사선 클립 코너 패널.
- **금지**: 라운드 큰 카드, 파스텔, 흰 배경, 부드러운 그림자, 세리프.
- **크리틱 루프**: ①글로우 과해 수치 가독성 해치나 ②시안·마젠타 외 색 샜나 ③관제 콘솔 긴장감이 3초 안에 오나 ④사선 클립·스캔라인 시그니처 있나.

## STYLE 02 — 공공데이터 포털
- **매력 축**: 장식을 뺀 신뢰. "이건 공식 통계다"라는 무게. 화려함의 반대가 주는 안정감.
- **팔레트**: 배경 #F5F6F8 / 기관 네이비 #00447C / 하늘색 표머리 #E8F1F9 / 텍스트 #1A1A1A / 보더 #D5DBE1
- **타이포**: Pretendard 단일. 굵기 대비로만 위계.
- **연출**: **최소.** 표가 위→아래 조용히 순차 렌더 정도. 카운트업·바운스·글로우 전부 금지. 정적이 미덕.
- **시그니처**: 표 하단 "※ 출처·집계 기준: …" + 짝수행 줄무늬 + 상단 기관 헤더 바 + 좌측 4px 네이비 보더 섹션 제목.
- **금지**: 그림자, 그라디언트, 이모지, 카운트업, 둥근 모서리, 과한 애니메이션.
- **크리틱 루프**: ①관공서 문서로 착각될 만큼 신뢰감 있나 ②튀는 장식 1개라도 있으면 제거 ③표가 명료히 읽히나 ④애니메이션이 조용한가(카운트업 없나).

## STYLE 03 — 트렌디 SaaS  `(v2 — 고도화 표준 ✅)`
- **매력 축**: "예쁜 정적 화면"이 아니라 "살아있는 제품". 2025 실리콘밸리(Linear/Vercel)의 가볍고 낙관적 감성.
- **팔레트**: 배경 #FBFAFF / 바이올렛 #7C5CFF(주) / 핑크 #FF8FC0 / 민트 #5EE6C8 / 잉크 #17143A / 보조 #7B7897. 그라디언트는 바이올렛→핑크 한 방향만(무지개 금지).
- **타이포**: 위계 극단화 — 최대(KPI 수치 40px+, 800, -0.03em, tabular-nums) vs 최소(캡션 11~12px) 크기비 4배+. 그라디언트 텍스트는 제품명 1곳.
- **깊이 3층**: L0 배경(#FBFAFF + 블러 90px 파스텔 블롭 2개) / L1 흰 카드(radius 24~28px, 0 10px 30px rgba(23,20,58,.06)) / L2 hover 부유(-4px + 컬러 틴트 0 20px 40px rgba(124,92,255,.15)) — 깊이 변화엔 색 동반.
- **연출 (로드 시퀀스 ~1.2초)**: ①헤더 자리잡음 ②KPI 4장 60ms 스태거로 떠오름 ③동시에 수치 0→목표 카운트업(800ms ease-out) ④차트 막대 왼쪽부터 40ms 간격 차오름 ⑤피크만 바운스+그라디언트 점등.
- **마이크로 인터랙션(4)**: KPI hover 부유 / 막대 hover 점등+라벨 볼드 / 진행중 태그·실시간 칩 점 2초 숨쉬기 / 행 hover 연보라+ID 진해짐.
- **시그니처**: 살아있는 숫자 — 카운트업 + 숨쉬는 실시간 인디케이터.
- **금지**: 직각, 격자 보더 테이블, 다크 배경, 모노스페이스, 무지개 그라디언트, 0.3초 넘는 개별 트랜지션, 무한 회전.
- **안전장치**: reduced-motion 시 연출 제거+최종 즉시 / JS 실패해도 최종 숫자 HTML에 존재(JS는 연출만) / hover는 focus-visible도.
- **크리틱 루프**: ①로드 후 3초간 시선 머무나 ②가장 밋밋한 1곳→디테일 추가 ③가장 과한 1곳→제거 ④색 팔레트 안·시퀀스 타이밍 일치.

## STYLE 04 — 고밀도 터미널
- **매력 축**: 픽셀당 정보량. 전문가가 "한 화면에 다 본다"는 효율. Bloomberg 단말기 밀도.
- **팔레트**: 배경 #0C0E12 / 패널 #111419 / 앰버 #F5A623(강조) / 그린 #3DDC84 / 레드 #FF5252 / 블루 #4DA3FF
- **타이포**: IBM Plex Mono 중심(11~12px), 한글만 Pretendard, 레이블 대문자+자간.
- **연출**: 값이 실시간 tick 하듯 미세 갱신(살짝 깜빡), 로그가 아래에서 한 줄씩 쌓임. 화려한 등장 X — 정보 우선, 연출은 "살아있다" 신호만.
- **시그니처**: 타임스탬프 이벤트 로그 패널 + 하단 F1~F5 펑션키 바 + 1px 3열 분할 + 수평 스트라이프 바 차트.
- **금지**: 여백 많은 레이아웃, 큰 폰트, 라운드, 파스텔, 장식적 그림자.
- **크리틱 루프**: ①한 화면에 꽉 찼나(여백 낭비 없나) ②11~12px 모노 밀도 유지되나 ③로그·펑션키 바 시그니처 있나 ④색이 기능적으로만 쓰였나.

## STYLE 05 — 브루탈리즘
- **매력 축**: 규칙을 아는 사람이 규칙을 깨는 자신감. 투박함이 곧 개성.
- **팔레트**: 배경 #FFFFFF / 잉크 #000000 / 원색 #FF3B00, #0037FF, #FFE600 (3색 이내)
- **타이포**: 제목 Archivo Black 과감한 크기, 본문 Pretendard.
- **연출**: 요소가 **딱딱 끊겨 하드하게** 등장(이징 없이 즉시/살짝 어긋나며). 클릭 시 하드 섀도 토글. **부드러운 바운스·스프링 금지.**
- **시그니처**: KPI 숫자를 화면에서 가장 큰 타이포로(숫자가 곧 레이아웃) + 오프셋 하드 섀도(8px 8px 0 #000) + 요소 -1~1deg 회전.
- **금지**: 그라디언트, 블러, 부드러운 그림자, 파스텔, 얇은 보더, 부드러운 이징.
- **크리틱 루프**: ①투박함이 "의도된" 것으로 읽히나(못 만든 게 아니라) ②하드 섀도·굵은 보더 있나 ③부드러운 모션 샜으면 제거 ④숫자가 가장 큰 요소인가.

## STYLE 06 — 글래스모피즘
- **매력 축**: 컬러풀한 배경 위 떠 있는 유리. 깊이와 투명함의 세련됨.
- **팔레트**: 배경 #4F46E5 → #9333EA → #EC4899 대각 그라디언트 / 유리 rgba(255,255,255,.14) / 텍스트 #FFF, 보조 rgba(255,255,255,.65)
- **타이포**: Pretendard 500~700 (밝은 배경 대비).
- **연출**: 로드 시 유리가 흐림(blur 40px)→선명(20px)으로 초점 맞춰지며 나타남, 배경 그라디언트가 아주 천천히 흐름.
- **시그니처**: 차트 막대도 반투명 유리 재질(배경 그라디언트 비침) + backdrop-filter blur(20px) + 1px rgba(255,255,255,.3) 보더 + 내부 하이라이트.
- **금지**: 불투명 흰 카드, 검정 텍스트, 직각, 격자 보더.
- **크리틱 루프**: ①유리 뒤로 배경이 실제 비치나(가짜 반투명 X) ②텍스트가 밝은 배경에서 읽히나 ③차트도 유리 재질인가 ④blur·투명도 일관.

## STYLE 07 — 미니멀 모노크롬
- **매력 축**: 뺄셈의 미학. 여백과 크기만으로 위계를 만드는 절제된 자신감(무인양품/스위스).
- **팔레트**: 배경 #FFF / 텍스트 #111 / 회색 #767676, #E5E5E5 / 포인트 #FF4D00 단 한 곳(피크·경고에만, 전체 3곳 이하)
- **타이포**: Pretendard. 크기 대비 극단(KPI 56px vs 레이블 11px), 자간 -0.03em.
- **연출**: 페이드 + 수치 카운트업만(절제). **블롭·컬러 모션·과한 등장 금지.** 여백이 주인공.
- **시그니처**: 색이 아닌 "여백과 크기"가 위계 — 1px 헤어라인과 공간만으로 구획, radius 0.
- **금지**: 카드 박스 남발, 두 색 이상 유채색, 그림자, 아이콘·이모지, 블롭.
- **크리틱 루프**: ①포인트 컬러 3곳 이하인가 ②여백이 충분히 넓나(답답하지 않나) ③그림자·박스 샜으면 제거 ④크기 대비만으로 위계 읽히나.

## STYLE 08 — 럭셔리 다크
- **매력 축**: 어둠 속 절제된 골드. "아무나 못 본다"는 프라이빗 뱅킹의 무게.
- **팔레트**: 배경 #0E0D0B / 패널 #171512 / 골드 #C9A96A(강조) / 아이보리 #EDE8DF / 보조 #8A8477
- **타이포**: 제목·수치 Cormorant Garamond(세리프, 가벼운 굵기, 큰 크기) / 본문 Pretendard.
- **연출**: **느리고 우아하게**(600~800ms, 부드러운 ease). 골드 헤어라인이 그려지듯, 수치가 천천히 페이드. **바운스·빠른 모션·카운트업 튐 금지.**
- **시그니처**: KPI 수치를 세리프체로 우아하게 — 대시보드가 아니라 자산 리포트처럼. 1px 골드 헤어라인 보더, 넉넉한 패딩, radius ≤4px.
- **금지**: 네온, 원색, 이모지, 알약 태그, 과한 글로우, 빠른/튀는 모션.
- **크리틱 루프**: ①세리프 수치가 우아한가(리포트 톤) ②골드가 절제됐나(과하지 않나) ③모션이 느리고 부드러운가 ④여백이 넉넉한가.

## STYLE 09 — 레트로 픽셀
- **매력 축**: 8비트 향수. 도트 그래픽 시대의 놀이 감성.
- **팔레트**: 배경 #1A1C2C / 패널 #29366F / 형광 그린 #38B764 / 옐로 #FFCD75 / 레드 #B13E53 / 텍스트 #F4F4F4
- **타이포**: 제목·수치 'Press Start 2P'(작게 절제), 한글 본문 Pretendard.
- **연출**: 카운트업이 **계단식**(정수 튀듯, 부드러운 보간 X), 픽셀 블록이 아래에서 한 칸씩 쌓임, 딱딱한 8비트 타이밍(step 이징).
- **시그니처**: 차트 막대를 픽셀 블록 쌓기 + 상태 뱃지를 게임 아이템 창처럼 + 픽셀 계단식 보더(box-shadow) + 디더링 배경.
- **금지**: 그라디언트, 블러, 안티앨리어싱 곡선, 세리프, 부드러운 이징.
- **크리틱 루프**: ①픽셀 계단(보더·모션)이 실제 각졌나(곡선 샜나) ②Press Start 2P 절제됐나(가독성) ③차트가 블록 쌓기인가 ④색이 8비트 팔레트 안인가.

## STYLE 10 — 에디토리얼 매거진
- **매력 축**: 읽는 대시보드. 종이 경제지의 데이터 저널리즘 — 숫자에 서사가 붙음.
- **팔레트**: 종이 #FAF7F2 / 잉크 #1C1B18 / 진홍 #A32638 / 헤어라인 #D8D2C6
- **타이포**: 제목 Noto Serif KR(700, 큼) / 본문 Pretendard / 수치 세리프로 크게.
- **연출**: 텍스트가 조판되듯 페이드인(위→아래 순차), 드롭캡 강조. 과한 모션 X — 지면은 정적, 읽힘 우선.
- **시그니처**: KPI를 "리드 문장"처럼 — 숫자 옆 헤드라인 톤 한 줄 해설 + 신문식 컬럼 그리드 + 2px+1px 이중 괘선.
- **금지**: 카드 그림자, 알약 태그, 네온, 다크 배경, 과한 애니메이션.
- **크리틱 루프**: ①"읽는" 지면 같나(그냥 대시보드 아니라) ②세리프 헤드라인·해설이 기사 톤인가 ③이중 괘선·컬럼 그리드 있나 ④진홍 포인트 절제됐나.

## STYLE 11 — 네이처 오가닉
- **매력 축**: 흙·잎·종이의 따뜻함. 기계적이지 않은, 손길이 느껴지는 자연스러움.
- **팔레트**: 배경 #F4F1E8 / 딥그린 #2D4A3E(주) / 세이지 #8BA888 / 테라코타 #C4704B(경고) / 잉크 #2B2A26
- **타이포**: 제목 Gowun Batang(세리프) / 본문 Pretendard, 따뜻한 굵기.
- **연출**: 요소가 **아래에서 자라나듯** 부드럽게(식물 성장 곡선, 곡선 이징), 잎 펴지듯. 급하지 않게.
- **시그니처**: 차트 막대를 식물 줄기처럼 끝을 둥글게 + 피크만 잎사귀 색 + 비대칭 유기적 radius(24px 40px 24px 40px) + 종이 질감 + 손그림 느낌 1.5px 보더.

## STYLE 12 — 뉴모피즘 소프트
- **매력 축**: 눌리고 튀어나온 촉감. 화면인데 만지고 싶은 물리적 표면.
- **팔레트**: 배경 #E4E9F2 단일 / 텍스트 #3D4A5C / 포인트 블루 #4A7CFF / 위험 #E36868
- **타이포**: Pretendard 600~700 (대비는 그림자가 만드므로 색 대비 낮게).
- **연출**: 표면이 눌리고 튀어나오는 촉감 — 로드 시 카드가 배경에서 볼록 솟음, hover 시 눌림(inset 전환). 부드러운 전환.
- **시그니처**: KPI 카드는 볼록(이중 그림자: 밝은 좌상단 -8px -8px 16px #FFF, 어두운 우하단 8px 8px 16px #C3CAD6), 차트 트랙·테이블은 오목(inset) — 요철 대비, radius 20px.
- **금지**: 강한 색 대비, 보더 라인, 다크 배경, 플랫 카드.
- **크리틱 루프**: ①볼록/오목 요철이 실제 보이나(플랫 아닌가) ②이중 그림자 있나 ③색 대비 낮게 유지되나 ④hover 시 눌림 촉감 있나.
