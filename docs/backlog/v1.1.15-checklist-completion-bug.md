# v1.1.15 후보 — Skill Checklist 마지막 task 가 in_progress 로 영구 잔존하는 버그

> 작성: 2026-05-10
> 우선순위: 중간 — UX 버그 (사용자가 "이전 자동 완료 task 안 사라짐" 으로 catch). 데이터 손상은 아님.

## 문제

js-super 플러그인을 다운스트림 프로젝트에서 사용 시 사용자 task 목록 상단에 다음과 같은 잔존 task 들이 영구적으로 표시:

```
◼ 9. designing-direction 자동 진입
◼ D10. /write-plan 자동 invoke OR 종료
```

브레인스토밍 → designing-direction → writing-plans 흐름이 끝났는데도 위 task 들이 in_progress 상태로 남아 있음.

## 근본 원인

여러 skill 의 Checklist 마지막 item 들이 **"다음 skill 을 Skill tool 로 invoke"** 패턴인데, 에이전트가 invoke 호출 직전에 해당 TaskCreate 항목을 `completed` 로 마킹하지 않고 새 skill 에 컨텍스트를 넘김. 결과적으로 기존 task 가 in_progress 로 잔존.

skill 본문이 _"you MUST create TaskCreate for each item and complete them in order"_ 라고 룰만 명시하고, **"invoke 직전에 마킹 완료해라"** 라는 명시적 reminder 가 없어서 에이전트가 transition 시점에 체크리스트 마무리를 잊음.

## 해당 위치

- `skills/brainstorming/SKILL.md:35` — item 9: "Auto-proceed to designing-direction"
- `skills/designing-direction/SKILL.md:26-27` — items 9+10:
  - 9: "Ask proceed-to-writing-plans gate"
  - 10: "On approval → invoke writing-plans via Skill tool. On hold → exit"
  - **두 item 이 사실상 같은 action (gate Q + answer handler)** 인데 쪼개져 있어 둘 다 잔존 위험.
- `skills/writing-plans/SKILL.md:28` — item 11: "Hand off to /execute-plan"

## 해결안 (Option A — 룰 강제)

각 skill 의 Checklist 끝에 한 줄 명시적 reminder 추가:

```markdown
If you find yourself skipping ahead, stop and create the missing task.

**Before invoking the next skill via Skill tool, mark ALL checklist TaskCreate items as completed (in_progress → completed). The Skill tool transition does NOT auto-complete prior tasks.**
```

## 해결안 (Option B — 구조 통합)

`designing-direction/SKILL.md` 의 items 9+10 을 1 item 으로 통합:

```markdown
9. **Ask proceed-to-writing-plans gate (v1.1.12+ — restored)** — change-history 직후 사용자에게 명시적 yes/no 게이트. On `yes` → invoke writing-plans via Skill tool. On `no` → exit with notice telling the user to run /write-plan later.
```

쪼개진 이유 없음 — semantic redundancy. 통합으로 TaskCreate 1개만 생성됨.

## 권장 — A + B 둘 다

- B 로 designing-direction 의 명시적 redundancy 제거 (TaskCreate 자체를 1개로 줄임)
- A 로 모든 transition skill 에 reminder 추가 (brainstorming 9 / designing-direction 통합 9 / writing-plans 11 모두)
