## 규칙

- 파일명: `v<버전>-<슬러그>.md` — 어느 버전쯤에 할지를 앞에 붙입니다.
- 프론트매터에 `status: backlog` · `priority` · `labels` 를 답니다.
- 도구를 쓰다가 발견한 문제면 `labels` 에 `dogfood-finding` 을 답니다.
- **구현하고 나면 여기서 지웁니다.** 그 기록은 `docs/features/<날짜>-<슬러그>/` 의
  산출물과 각 문서 하단 `## 변경이력` 에 남으니, 백로그에 남겨두면 이중 관리가 됩니다.
